Comment lancer l'application Java GestionFormations

Pré-requis:
	    - Installer le JRE 8 (ou superieur) sur la machine cible, 

		    (Téléchargé depuis: http://www.oracle.com/technetwork/java/javase/downloads/index.html).

	    - Machine cible dote d'un systeme d'exploitation Windows.
	    - modifier, si nécessaire, vos Variables d'environnement pour pouvoir exécuter les commandes JAVA dans le terminal windows.

Execution :
	    - Copier le fichier ProxiBanqueV1.jar et launch.bat (ou le dossier les contenant) en local sur la machine,
	    - Executez launch.bat pour une excution immédiate en ligne de commande ou bien,

	    - Copier le path du répertoire parent contenant le fichier .jar,
            - Ouvrir un invite de commande,
	    - Taper "cd <path>" puis pressez entrée (avec <path> le path copié precedemment),
	      
			Ex: "cd C:\formationJava\livraison".

	    - Taper "java -jar ProxiBanqueV1.jar" et presser entrée. 

Consulter la documentation: 
	    - Double-cliquer sur le fichier index.html se trouvant dans le repertoire "doc".
